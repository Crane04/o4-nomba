# O4 Roadmap

O4 is currently a functional MVP focused on Nomba virtual account reconciliation.

## Completed

- Invoice creation
- Virtual account transfer tracking
- Nomba webhook handling
- Manual transfer sync
- Confidence-based matching
- Automatic reconciliation for strong matches
- Flagged payment review queue
- Manual confirmation and rejection

## Planned

- Multi-bank support
- More advanced reconciliation rules
- Team permissions
- Reconciliation analytics
- Accounting integrations
- API access for businesses
- Exportable reports
- Better webhook retry and replay tooling

## Long-Term Direction

O4 can become a provider-agnostic reconciliation layer for businesses that collect bank transfers across multiple banks, processors, and accounting systems.
